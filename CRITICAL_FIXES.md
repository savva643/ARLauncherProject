# Критические исправления

## Исправлено:

1. **Логирование IMU** - вернул логи каждые 60 кадров с полной информацией 6DOF (позиция, ротация, акселерометр, гироскоп)

2. **Движение камеры** - камера теперь обновляется напрямую в колбэке LensEngine без задержки интерполяции

3. **Видеофон** - добавлен метод `renderStoredVideoBackground()` который рендерит сохраненную видео текстуру в каждом кадре

4. **Объекты** - исправлен stride в vertex attributes (было 6, стало 8 для поддержки UV координат)

5. **Кодировка текста** - улучшена обработка UTF-8, но для полной поддержки русских символов нужна библиотека FreeType

## Что нужно проверить:

- Видеофон должен отображаться (вызывается renderStoredVideoBackground в render())
- Камера должна двигаться плавно (обновляется в колбэке)
- Логи должны показывать 6DOF данные каждые 60 кадров
- Объекты должны отображаться правильно (исправлен stride)
