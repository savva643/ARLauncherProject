# Что отображает ARLauncher сейчас

## Последовательность отображения:

### 1. Splash экран (0-9 секунд):

#### 0-3 секунды:
- **Экран**: Черный фон
- **Текст "Spatial Home"**: Появляется по центру экрана с fade-in анимацией
- **Текст "GlaskiOS"**: Появляется внизу по центру с fade-in анимацией
- **Видео/3D/UI**: Hidden (opacity = 0)

#### 3-6 секунд:
- **Видео-фон**: Fade-in, RGB поток с камеры iPhone
- **3D мир**: Кубы и сфера проявляются синхронно с видео
- **3D UI окна**: Появляются плавающие окна и кнопки с эффектом стекла
- **Текст**: Splash надписи остаются поверх

#### 6-9 секунд:
- **Текст**: Splash надписи плавно исчезают (fade-out)
- **Сцена**: Видео, 3D мир и 3D UI остаются видимыми

### 2. Основной режим (после 9 секунд):

#### RGB Видео-фон:
- Источник: Камера iPhone через SensorConnector
- Разрешение: 720x540 / 720x960
- Отрисовка: Шейдер `video` + полноэкранный VAO, поддержка opacity

#### 3D Виртуальный мир:
- Кубы: Несколько вращающихся примитивов (Y-axis spin)
- Сфера: Вращение по X/Y, размещена выше и дальше
- Рендеринг: Шейдер `basic` с освещением и depth-test

#### Плавающий 3D UI (glassmorphism):
- Окно "Command Center" (1.2 m x 0.6 m) с подсветкой и стеклянным эффектом
- Кнопки "Create Window", "Add 3D Model", "Quick Menu" (0.6 m x 0.35 m)
- Текстовая панель "AR Launcher Ready" (0.8 m x 0.25 m)
- Отрисовка: simple_nvg -> текстура -> 3D квад через шейдер `ui`
- Размещение: В мировой системе координат, направлено лицом к камере (billboard)

#### Камера AR:
- Синхронизация: Pitch/Roll из IMU, позиция пока (0,0,0)
- Обновление: Каждый кадр через `Scene::updateCameraFromAR`

## Технические детали:

### Рендеринг:
- API: OpenGL 3.3+
- Шейдеры: `basic`, `video`, `ui`, `glassmorphism`
- FBO: создаются для каждого UI окна, текстуры RGBA
- simple_nvg: Минимальный NanoVG-совместимый рендер для текста и панелей

### Производительность:
- Цель: 60 FPS
- Оптимизации: VAO/VBO, шейдеры, отказоустойчивый fallback

### Логирование:
- IMU 6DoF (каждые 60 кадров)
- Акселерометр, гироскоп, гравити, вычисленные pitch/roll

## Что работает:
- Видео-фон, 3D мир, splash анимация
- 3D UI окна/кнопки с glassmorphism (через FBO + simple_nvg)
- `ARLauncherAPI` для создания окон, кнопок, текста и моделей
- IMU привязка камеры, логирование каждые 60 кадров
- Полное копирование шейдеров в build

## Запуск:
```bash
cd ~/Documents/ARLauncherProject
./build_linux.sh
cd ARLauncher/build/bin
./ARLauncher
```

## Итоговый результат:
- Реальный видео-фон + виртуальный мир + 3D интерфейс в едином слое
- Плавающие окна, кнопки и текст в пространстве (VisionOS-like)
- Готовый публичный API для стороннего контента
- Полностью автоматизированная последовательность запуска (splash -> AR сцена)
